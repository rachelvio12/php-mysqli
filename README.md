# Project Ujian Pemrograman Web

## Aplikasi Pengelolaan Data Mahasiswa (PHP & MySQL)

Aplikasi ini merupakan hasil dari **Ujian Pemrograman Web** yang dibuat menggunakan **PHP Native**, **MySQL**, dan **Bootstrap 5**. Aplikasi berfungsi untuk mengelola data mahasiswa dengan fitur CRUD (Create, Read, Update, Delete) serta dilengkapi dengan fitur upload foto.
